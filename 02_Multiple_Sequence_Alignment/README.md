# Multiple Sequence Alignment of Lactobacillus 16S rRNA Sequences

## Overview

This project compares the 16S ribosomal RNA (16S rRNA) sequences of five Lactobacillus species using multiple sequence alignment.

The objective is to identify conserved and variable regions and visualize sequence conservation using a WebLogo.

(This project was reconstructed from my undergraduate Bioinformatics coursework.)
---

## Species Included

-1.Lactobacillus helsingborgensis
(NCBI Reference Sequence: NR_126253.1)
-2. Lactobacillus melliventris
(NCBI Reference Sequence: NR_126252.1)
-3. Lactobacillus kullabergensis
(NCBI Reference Sequence: NR_126251.1)
-4. Lactobacillus kimbladii
(NCBI Reference Sequence: NR_126250.1)
-5. Lactobacillus ultunensis
(NCBI Reference Sequence: NR_042802.1)
