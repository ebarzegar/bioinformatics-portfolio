# Dot Plot and UniProt Analysis of Human TPI1

## Project Overview

This project explores the human **Triosephosphate Isomerase (TPI1)** gene and protein using several bioinformatics tools. Dot plot analysis was performed to compare nucleotide and protein sequences, while UniProtKB was used to investigate protein function, post-translational modifications (PTMs), and structural information.

The project demonstrates how sequence comparison and protein annotation can be combined to better understand the biological characteristics of a protein.

---

## Objectives

- Compare the nucleotide sequences of human and rabbit **TPI1** using EMBOSS Dotmatcher.
- Identify repeated regions in the human TPI1 protein sequence using UGENE.
- Explore Gene Ontology (GO) annotations in UniProtKB.
- Identify post-translational modifications (PTMs).
- Determine whether the protein contains disulfide bonds.
- Visualize the three-dimensional (3D) protein structure.

---
