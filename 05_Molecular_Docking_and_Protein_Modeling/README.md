# Project 5: Molecular Docking and Protein Modeling

## Overview

This project explores molecular docking, protein interaction networks, and comparative protein structure modeling using publicly available bioinformatics tools.

## Objectives

- Perform molecular docking using SwissDock.
- Explore protein-protein interaction networks using STRING.
- Predict a modified protein structure using SWISS-MODEL.

## Tools

- SwissDock
- STRING
- SWISS-MODEL
